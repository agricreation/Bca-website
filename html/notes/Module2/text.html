Displaying Text in List

In order to display text in list there are two types of lists:
1.Ordered List
2.Unordered List
In the following list we can use the <li> tag to insert text and to display text in the lists.
Example:

<ol or ul>
<li>Item number 1</li>
<li>Item number 2</li>
<li>Item number 3</li>
<li>Item number 4</li>
</ol or ul>
